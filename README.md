# Ajna bouding boxes

Treinamento de modelo para detecção das bordas do contêiner em imagens de escaneamento


Para iniciar:

```
# git clone https://github.com/IvanBrasilico/ajna_bbox
# python3 -m venv venv
# pip install -r requirements.txt
```

